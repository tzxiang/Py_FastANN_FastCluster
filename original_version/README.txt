This is the original version of FastANN and Fastcluster used for approximate nearest neighbours and 
fast distributed clustering, released in 2009.

Web: https://www.robots.ox.ac.uk/~vgg/software/fastanncluster/


FASTANN: A library for fast approximate nearest neighbours (C++ with python interfaces)    
FASTCLUSTER: A library for fast, distributed clustering (python library)
